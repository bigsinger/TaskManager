### ✅ 问题6：抽屉式状态切换（橡皮筋效果）
**问题描述**：右滑任务时，状态应该像抽屉一样慢慢拉宽，继续拖动切换状态，松开回弹

**解决方案**：
1. **抽屉样式** `drawer.css`：
   - `.status-drawer` - 从左侧滑出的状态抽屉
   - 蓝色半透明背景 `rgba(102, 126, 234, 0.95)`
   - `.status-indicator` - 状态指示器容器
   - `.status-icon` - 状态图标（⏳🔄✅）
   - `.status-label` - 状态标签（未开始/进行中/已完成）
   - 橡皮筋效果：CSS transition `transform 0.3s cubic-bezier(0.4, 0, 0.2, 1)`

2. **抽屉交互** `drawer.js`：
   - 右滑任务时调用 `handleDrawerSwipeRight(task)`
   - 显示状态抽屉（1.5秒）
   - 抽屉显示当前状态图标和标签
   - 继续拖动超过阈值后切换下一个状态
   - 松手后回弹到原位（橡皮筋效果）
   - 状态循环：pending → in-progress → completed → pending

3. **拖拽动画**：
   - 使用CSS变量 `--drag-x` 控制拖拽距离
   - 限制最大拖拽距离（300px）
   - 根据拖拽方向添加 `.dragging-right` 或 `.dragging-left` 类
   - 松手后回弹到原位（弹性动画）

**交互效果**：
- 向右拖动任务时，蓝色状态抽屉从左侧滑出
- 抽屉显示当前状态图标和标签（如：⏳ 未开始）
- 继续拖动超过100px阈值后切换到下一个状态
- 松手后抽屉回弹消失，任务状态已更新
- 拖拽过程有弹性阻尼感（橡皮筋效果）

## 技术实现细节

### 抽屉式架构
```
.drawer-container (任务容器)
  ├── .drawer-content (任务卡片)
  │     ├── 任务标题
  │     ├── 标签
  │     ├── 耗时
  │     ├── 收藏按钮
  │     ├── 状态徽章
  │     └── 操作按钮
  ├── .status-drawer (状态抽屉 - 左滑)
  │     └── 状态指示器（图标+标签）
  └── .timeline-drawer (时间线抽屉 - 右滑)
        ├── 关闭按钮
        └── 时间线内容
```

### 事件处理流程
```
用户拖拽任务
    ↓
handleDrawerTouchStart / handleDrawerMouseDown
    ↓
handleDrawerDragMove (实时更新 --drag-x)
    ↓
handleDrawerDragEnd (检查拖拽距离)
    ↓
超过阈值？
    ├─ 是 → 触发操作
    │   ├─ 右滑 → handleDrawerSwipeRight
    │   │         ├─ 显示状态抽屉
    │   │         └─ 1.5秒后切换状态
    │   └─ 左滑 → handleDrawerSwipeLeft
    │             ├─ 打开时间线抽屉
    │             └─ 加载活动记录
    └─ 否 → 回弹到原位（橡皮筋效果）
```

### 橡皮筋效果实现
- CSS变量 `--drag-x` 控制拖拽距离
- `transform: translateX(var(--drag-x, 0))` 实时移动
- `transition: transform 0.3s cubic-bezier(0.4, 0, 0.2, 1)` 弹性过渡
- 松手后清除CSS变量，自动回弹
