## 🚀 快速开始

### 1. 访问系统

打开浏览器，访问 TaskManager 系统地址：
- 本地开发环境: http://localhost:3001
- 生产环境: https://taskmanager.example.com

### 2. 登录系统

#### 2.1 账号密码登录

1. 点击"登录"按钮
2. 输入邮箱和密码
3. 点击"登录"按钮

#### 2.2 GitHub 登录

1. 点击"使用 GitHub 登录"按钮
2. 授权 GitHub 应用
3. 自动创建账号并登录

#### 2.3 Google 登录

1. 点击"使用 Google 登录"按钮
2. 选择 Google 账号
3. 自动创建账号并登录

### 3. 首次使用

首次登录后，系统会引导您完成以下步骤：
1. 完善个人信息（昵称、头像）
2. 创建第一个情境（项目）
3. 创建第一个任务

## 📚 功能指南

### 1. 任务管理

#### 1.1 创建任务

**方式一：快速创建**
1. 点击页面右上角的"新建任务"按钮
2. 输入任务标题
3. 选择任务状态（待处理/进行中/已完成）
4. 点击"保存"按钮

**方式二：详细创建**
1. 点击"新建任务"按钮
2. 填写任务信息：
   - 标题（必填）
   - 描述（可选）
   - 状态（待处理/进行中/已完成）
   - 优先级（0-10）
   - 标签（可选）
   - 预计耗时（可选）
   - 执行者（可选）
   - 报告人（可选）
   - 验证人（可选）
3. 点击"保存"按钮

**方式三：使用模板**
1. 点击"新建任务"按钮
2. 选择"使用模板"
3. 从预设模板中选择
4. 根据需要修改内容
5. 点击"保存"按钮

#### 1.2 查看任务

**任务列表视图**
- 显示所有任务的卡片
- 支持按状态、标签、优先级筛选
- 支持搜索任务
- 支持按创建时间、更新时间排序

**任务详情视图**
- 点击任务卡片进入详情页
- 显示任务完整信息
- 显示任务时间线
- 显示任务评论

#### 1.3 编辑任务

1. 点击任务卡片上的"编辑"按钮
2. 修改任务信息
3. 点击"保存"按钮

#### 1.4 删除任务

1. 点击任务卡片上的"删除"按钮
2. 确认删除操作

**注意**：删除操作不可恢复，请谨慎操作。

#### 1.5 任务状态管理

**切换任务状态**
- 方式一：在任务详情页修改状态
- 方式二：在任务列表页使用手势滑动（向右滑动切换状态）
- 方式三：使用快捷键（Ctrl + 1/2/3）

**状态说明**
- **待处理 (Pending)**: 任务尚未开始
- **进行中 (In Progress)**: 任务正在进行
- **已完成 (Completed)**: 任务已完成

#### 1.6 任务收藏

**收藏任务**
- 点击任务卡片上的五角星图标
- 收藏后图标变为黄色

**取消收藏**
- 再次点击五角星图标
- 图标变回灰色

**筛选收藏任务**
- 在任务列表页点击"收藏"筛选器
- 只显示已收藏的任务

#### 1.7 任务标签

**添加标签**
1. 在创建/编辑任务时，在"标签"输入框中输入标签名
2. 按回车键添加标签
3. 可以添加多个标签

**删除标签**
1. 点击标签上的"×"按钮
2. 标签被删除

**按标签筛选**
1. 在任务列表页点击标签筛选器
2. 选择要筛选的标签
3. 只显示包含该标签的任务

#### 1.8 任务搜索

**搜索任务**
1. 在搜索框中输入关键词
2. 系统自动搜索匹配的任务
3. 支持搜索任务标题、描述、标签

**高级搜索**
- 支持多关键词搜索（空格分隔）
- 支持精确匹配（使用引号）
- 支持排除关键词（使用减号）

### 2. 情境（项目）管理

#### 2.1 创建情境

1. 点击情境切换器
2. 点击"新建情境"按钮
3. 填写情境信息：
   - 名称（必填）
   - 描述（可选）
   - 头像（可选）
   - 是否公开（可选）
4. 点击"保存"按钮

#### 2.2 切换情境

1. 点击情境切换器
2. 从列表中选择要切换的情境
3. 任务列表自动更新为当前情境的任务

#### 2.3 编辑情境

1. 点击情境切换器
2. 点击情境名称旁的"编辑"按钮
3. 修改情境信息
4. 点击"保存"按钮

#### 2.4 删除情境

1. 点击情境切换器
2. 点击情境名称旁的"删除"按钮
3. 确认删除操作

**注意**：删除情境会同时删除该情境下的所有任务，请谨慎操作。

2. 修改评论内容
3. 点击"保存"按钮

#### 3.4 删除评论

1. 点击评论上的"删除"按钮
2. 确认删除操作

### 4. 任务时间线

#### 4.1 查看时间线

1. 进入任务详情页
2. 滚动到"时间线"部分
3. 查看任务的所有操作历史

#### 4.2 时间线事件类型

- **创建**: 任务被创建
- **开始**: 任务开始执行
- **评论**: 添加了评论
- **编辑**: 任务被编辑
- **状态变更**: 任务状态发生变化
- **完成**: 任务被标记为完成
- **分配**: 任务被分配给用户
- **收藏**: 任务被收藏/取消收藏

### 7. 权限管理

#### 7.1 角色说明

**拥有者 (Owner)**
- 拥有全部权限
- 可以创建租户
- 可以管理所有资源

**任务执行者 (TaskExecutor)**
- 只能修改自己任务的状态
- 可以添加评论
- 不能修改任务内容

**项目管理员 (ProjectManager)**
- 可以修改任务状态
- 可以添加评论
- 可以分配任务
- 不能修改任务内容

#### 7.2 查看权限

1. 进入个人中心
2. 查看"我的角色"
3. 查看当前角色的权限

## ❓ 常见问题

### 1. 登录问题

**Q: 忘记密码怎么办？**

A: 点击登录页的"忘记密码"链接，输入邮箱，系统会发送重置密码的邮件。

**Q: 无法使用 GitHub/Google 登录？**

A: 请确保：
- 已正确配置 OAuth 应用
- 浏览器允许弹窗
- 网络连接正常

### 2. 任务管理问题

**Q: 如何批量操作任务？**

A: 在任务列表页勾选多个任务，然后点击批量操作按钮（批量完成、批量删除）。

**Q: 如何恢复已删除的任务？**

A: 目前不支持恢复已删除的任务，请谨慎操作。

**Q: 如何设置任务优先级？**

A: 在创建/编辑任务时，设置优先级（0-10），数字越大优先级越高。

### 3. 情境管理问题

**Q: 如何将任务移动到其他情境？**

A: 编辑任务，选择新的情境，保存即可。

**Q: 删除情境会删除任务吗？**

A: 是的，删除情境会同时删除该情境下的所有任务，请谨慎操作。

### 4. 权限问题

**Q: 为什么无法编辑某些任务？**

A: 请检查您的角色权限，任务执行者只能修改自己任务的状态。

**Q: 如何申请更高权限？**

A: 请联系租户管理员或拥有者，请求分配更高权限的角色。

### 5. 性能问题

**Q: 任务列表加载缓慢？**

A: 请尝试：
- 减少显示的任务数量
- 使用筛选功能
- 清除浏览器缓存

**Q: 如何提高系统性能？**

A: 建议：
- 使用现代浏览器（Chrome、Firefox、Edge）
- 定期清理浏览器缓存
- 关闭不必要的浏览器标签页
