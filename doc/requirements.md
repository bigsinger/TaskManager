## 项目概述

TaskManager 是一个企业级任务管理系统，支持任务的创建、编辑、删除、查看等基本功能，同时提供了丰富的企业级特性，包括主题切换、多语言支持、状态筛选、标签管理、分页等功能。

### 项目目标

- 提供简单易用的任务管理界面
- 支持企业级的功能需求
- 提供良好的用户体验
- 支持移动端和局域网访问

### 技术栈

#### 前端
- HTML5
- CSS3
- JavaScript (ES6+)
- Fetch API
- LocalStorage

#### 后端
- Node.js
- Express
- TypeScript
- Prisma
- SQLite

## 功能需求

### 1. 基础功能

#### 1.1 任务管理
- **新建任务**
  - 支持设置任务标题
  - 支持设置任务描述
  - 支持设置任务状态（pending/in-progress/completed）
  - 支持设置任务标签
  - 标签输入支持 Enter 键添加
  - 标签输入支持失去焦点自动添加
  - 标签支持点击 × 按钮删除
  - 防止重复添加标签

- **编辑任务**
  - 支持修改任务标题
  - 支持修改任务描述
  - 支持修改任务状态
  - 支持修改任务标签
  - 编辑时保持选中状态
  - 保存后保持选中状态

- **删除任务**
  - 支持删除任务
  - 删除时渐进式消失
  - 删除时不刷新页面
  - 删除后自动处理空页面

- **查看任务列表**
  - 支持查看所有任务
  - 支持查看任务详情
  - 支持查看任务状态
  - 支持查看任务标签
  - 支持查看任务创建时间

### 2. 主题设置

#### 2.1 主题切换
- 支持浅色主题
- 支持深色主题
- 支持跟随系统主题
- 主题设置持久化

#### 2.2 主题样式
- 浅色主题：白色背景，深色文字
- 深色主题：深色背景，浅色文字
- 系统主题：跟随系统设置

### 3. 多语言支持

#### 3.1 语言切换
- 支持英语
- 支持中文
- 语言设置持久化

#### 3.2 翻译覆盖
- 所有界面文本支持翻译
- 所有提示信息支持翻译
- 所有错误信息支持翻译

### 4. 标题和副标题

#### 4.1 标题显示
- 标题单独显示一行
- 标题固定为 "Task Manager"

#### 4.2 副标题显示
- 副标题默认为 "My Tasks"
- 副标题可通过设置修改
- 副标题持久化

### 5. 状态筛选

#### 5.1 筛选按钮
- 显示三个筛选按钮：Pending、In Progress、Completed
- 支持多选
- 选择后实时过滤任务列表
- 选中的按钮高亮显示

#### 5.2 筛选逻辑
- 未选择任何状态：显示所有任务
- 选择一个或多个状态：显示对应状态的任务
- 筛选与标签筛选协同工作

### 6. 分页功能

#### 6.1 分页信息
- 显示当前页范围（如 "1-20 of 100"）
- 显示总任务数

#### 6.2 分页导航
- 首页按钮
- 上一页按钮
- 下一页按钮
- 末页按钮
- 智能页码显示（最多5个页码，超出用省略号）

#### 6.3 每页数量
- 支持选择每页显示数量（5/10/20/50/100）
- 每页数量设置持久化

#### 6.4 按钮状态
- 当前页高亮显示
- 首页时禁用首页和上一页按钮
- 末页时禁用下一页和末页按钮

#### 6.5 响应式设计
- 支持移动端显示
- 支持不同屏幕尺寸

### 7. 标签功能

#### 7.1 标签输入
- 任务支持标签字段
- 新建和编辑时可以设置标签
- 标签输入（Enter键添加，失去焦点自动添加）
- 标签删除（点击×按钮）
- 防止重复添加

#### 7.2 标签显示
- 标签显示在任务列表中
- 标签显示为小标签样式
- 标签颜色统一

#### 7.3 标签云
- 显示所有标签
- 标签云根据频率显示不同大小（12px-24px）
- 标签云根据频率排序
- 标签云根据频率显示不同透明度（0.6-1.0）
- 标签云悬停效果
- 标签云选中功能
- 标签云选中状态与状态筛选按钮一致

#### 7.4 标签筛选
- 支持标签筛选
- 标签筛选与状态筛选协同工作
- 选中的标签高亮显示
- 未选中的标签透明显示

### 8. 用户体验优化

#### 8.1 动画效果
- 删除任务时渐进式消失
- 删除任务时向左滑动
- 删除任务时淡出
- 鼠标悬停高亮效果
- 点击选中高亮效果
- 流畅的过渡动画

#### 8.2 选中状态
- 点击任务时切换选中状态
- 编辑任务时保持选中状态
- 保存修改后恢复选中状态
- 新建任务时清除选中状态
- 关闭表单时清除选中状态

#### 8.3 视觉反馈
- 清晰的选中状态
- 清晰的悬停效果
- 清晰的按钮状态
- 清晰的加载状态

### 9. 移动端支持

#### 9.1 响应式设计
- 支持手机端访问
- 支持平板端访问
- 支持不同屏幕尺寸
- 自适应布局

#### 9.2 局域网访问
- 支持局域网访问
- 支持不同域名访问
- API URL 自动适配

### 10. 版本管理

#### 10.1 Git 版本控制
- Git 版本控制
- 完整的提交历史
- 可以随时回退

#### 10.2 版本标签
- 使用语义化版本号
- 创建版本标签
- 记录版本变更

## 开发规范

### 1. 代码规范
- 使用 ES6+ 语法
- 函数命名使用驼峰命名法
- 变量命名使用驼峰命名法
- 常量命名使用大写下划线
- 添加必要的注释

### 2. Git 规范
- 提交信息使用英文
- 提交信息格式：`类型: 描述`
- 提交类型：
  - `feat`: 新功能
  - `fix`: 修复bug
  - `docs`: 文档更新
  - `style`: 代码格式调整
  - `refactor`: 重构
  - `test`: 测试
  - `chore`: 构建/工具链

### 3. 版本规范
- 使用语义化版本号（Semantic Versioning）
- 格式：`MAJOR.MINOR.PATCH`
  - MAJOR: 不兼容的API修改
  - MINOR: 向下兼容的功能性新增
  - PATCH: 向下兼容的问题修正
