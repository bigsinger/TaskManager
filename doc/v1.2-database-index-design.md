## 索引列表

### 1. Task 表索引

- `@@index([userId])` - 用户 ID 索引
- `@@index([status])` - 状态索引
- `@@index([createdAt])` - 创建时间索引
- `@@index([status, createdAt])` - 复合索引

### 2. User 表索引

- `@@index([email])` - 邮箱索引（已存在唯一索引）
- `@@index([role])` - 角色索引
- `@@index([isActive])` - 激活状态索引

### 3. Tag 表索引

- `@@index([name])` - 名称索引
- `@@index([userId])` - 用户 ID 索引

### 4. AuditLog 表索引

- `@@index([userId])` - 用户 ID 索引
- `@@index([action])` - 操作索引
- `@@index([resource])` - 资源索引
- `@@index([createdAt])` - 创建时间索引
- `@@index([userId, createdAt])` - 复合索引
